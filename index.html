Código Descripción
git init Inicializa un nuevo repositorio de Git.
git add . Agrega todos los cambios al área de preparación (staging).
git commit -m "mensaje" Confirma los cambios con un mensaje descriptivo.
git push -u origin main Sube los cambios a Git
git status Muestra el estado actual de los cambios
