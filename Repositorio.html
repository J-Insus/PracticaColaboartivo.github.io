Pasos para crear y subir un repositorio a GitHub

1. Crear Repositorio en GitHub

Ir a GitHub y crear un nuevo repositorio. No olvides añadir un README si quieres.

2. Clonar Repositorio

Al presionar el boton verde "Code", muestra una URL (.github.io.git). Usa git clone "URL" para copiar el repositorio en tu computadora.

3. Ubicandose en la carpeta

Es importante estar dentro de la carpeta, por eso despues de clonar es importane cambiarse a la ubicacion de repositorio usando "cd"

4. Primer Commit

Despues de hacer cambios, y gaurdarlos todos ya se puede realizar un commit usando primero :
git add . y luego
git commit -m "Primer commit"

5. Subir a GitHub

Envía los cambios a GitHub con:
git push origin main, el "main" puede cambiar dependiendo el tu nombre de rama

6. ¿Cómo los puedo publicar?

Desde las configuraciones del repositorio, dirigete a la seccione de GitHub Pages .

7. Publicar en GitHub Pages

Asegurate que "Source este seleccionado == Deploy from a branch" y Branch toma la rama main.

8. Es común que suceda

Que no aparezca el dominio o el link a la pagina de nuestro repositorio para ver los resultados, asi que en estos casos cambiamos en branch entre /(root) y /docs hasta que apareza o en su defecto se actualicen los cambios nuevos
